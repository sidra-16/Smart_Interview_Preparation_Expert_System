# Smart Interview Preparation Expert System

An AI-powered interview readiness analyzer built using **Python, Streamlit, SWI-Prolog, and PySwip**.
The system evaluates technical interview preparation using symbolic AI techniques such as **Backward Chaining, Rule-Based Reasoning, Unification, and Explainable AI**.

---

## Features

* Role-specific interview readiness analysis
* Skill gap detection using Prolog backward chaining
* Weighted readiness scoring system
* Personalized study recommendations
* Explainable AI (XAI) feedback
* Interactive mock interview module
* Modern dark-themed Streamlit UI

---

## Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/smart-interview-expert-system.git
cd smart-interview-expert-system
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Install SWI-Prolog

Download and install SWI-Prolog:

https://www.swi-prolog.org/download/stable

---

## Run the Project

```bash
streamlit run app.py
```

---

## System Workflow

1. User selects target role and skills
2. Python sends queries to Prolog
3. Prolog identifies weak skills using backward chaining
4. Readiness score is calculated using rule-based logic
5. Recommendations and explanations are generated
6. User can attempt a mock interview
