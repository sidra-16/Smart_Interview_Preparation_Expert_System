# Smart Interview Preparation Expert System

A web-based application that helps users evaluate their interview preparation for different technical roles. The system analyzes user skills and provides readiness scores, skill gap analysis, recommendations, and mock interview practice.

## Features

* Skill gap analysis
* Readiness score calculation
* Personalized recommendations
* Mock interview practice

## Technologies Used

* Python
* Streamlit
* SWI-Prolog
* PySwip

## Supported Roles

* Software Engineer
* Web Developer
* Data Analyst
* UI/UX Designer

## Run the Project

```bash
streamlit run app.py
```
